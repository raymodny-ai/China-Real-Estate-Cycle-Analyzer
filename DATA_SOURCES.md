**版本**: v2.1  
**更新日期**: 2026-03-03

---

## 📋 项目所需数据总览

本项目需要以下 **四大核心指标**：

| 指标 | 英文名 | 数据库字段 | 数据频率 | 需要的原始数据 |
|------|--------|-----------|---------|--------------|
| **ACI** | Area Clearance Index 去化周期 | `aci` | 月度 | 库存面积 + 销售面积 |
| **FPI** | Financial Pressure Index 资金链压力 | `net_financing_cash_flow` | 年度/季度 | 房企融资现金流 |
| **LPR** | Land Premium Rate 土地溢价率 | `premium_rate` | 月度 | 土地成交价 + 起拍价 |
| **RPR** | Rent-to-Price Ratio 租售比 | `rent_price_ratio` | 月度 | 房价 + 租金 |

---

## 🗂️ 数据源推荐 (按优先级排序)

### 1. ACI 数据 - 去化周期

**计算公式**: `ACI = 可售库存面积 / 月均销售面积`

| 优先级 | 数据源 | 类型 | 费用 | 说明 |
|--------|--------|------|------|------|
| ⭐⭐⭐⭐ | **国家统计局** | 官方 | 免费 | https://data.stats.gov.cn 月度销售数据 |
| ⭐⭐⭐⭐ | **AKShare** | 开源 | 免费 | Python接口，调用国家统计局数据 |
| ⭐⭐⭐ | **中指研究院/云** | 商业 | 免费/付费 | 库存数据报告 |
| ⭐⭐⭐ | **克尔瑞 CRIC** | 商业 | 付费 | 56+城市数据 |
| ⭐⭐ | **贝壳研究院** | 商业 | 免费/付费 | 二手房数据 |
| ⭐ | **地方统计局** | 官方 | 免费 | 需要手动汇总 |

**AKShare 使用示例**:
```python
import akshare as ak

# 商品房销售面积
df = ak.macro_china_new_house()
# 房地产开发投资
df = ak.macro_china_real_estate_investment()
```

---

### 2. FPI 数据 - 房企资金链压力

**计算公式**: `FPI = 核心房企净融资现金流`

| 优先级 | 数据源 | 类型 | 费用 | 说明 |
|--------|--------|------|------|------|
| ⭐⭐⭐⭐ | **AKShare** | 开源 | 免费 | 上市公司财报数据 |
| ⭐⭐⭐ | **东方财富** | 商业 | 免费 | 财报数据接口 |
| ⭐⭐⭐ | **巨潮资讯网** | 官方 | 免费 | 财报公告 cninfo.com.cn |
| ⭐⭐⭐ | **新浪财经** | 商业 | 免费 | 财报数据接口 |
| ⭐⭐ | **Wind 万得** | 商业 | 付费 | 机构级最全 |
| ⭐⭐ | **Choice 东方财富** | 商业 | 付费 | 性价比高 |

**AKShare 使用示例**:
```python
import akshare as ak

# 房企融资数据
df = ak.stock_individual_info_em(symbol="000002")  # 万科
# 或者通过财报数据计算
```

---

### 3. LPR 数据 - 土地溢价率

**计算公式**: `LPR = (成交价 - 起拍价) / 起拍价 × 100%`

| 优先级 | 数据源 | 类型 | 费用 | 说明 |
|--------|--------|------|------|------|
| ⭐⭐⭐⭐ | **AKShare** | 开源 | 免费 | 土地成交数据 |
| ⭐⭐⭐ | **中指研究院/云** | 商业 | 免费/付费 | 月度土拍报告 |
| ⭐⭐⭐ | **克尔瑞 CRIC** | 商业 | 付费 | 土拍数据 |
| ⭐⭐ | **土拍网** | 商业 | 付费 | tupai.com |
| ⭐⭐ | **公共资源交易中心** | 官方 | 免费 | 各省市交易网站 |
| ⭐ | **自然资源部** | 官方 | 免费 | 全国汇总数据 |

**AKShare 使用示例**:
```python
import akshare as ak

# 土地成交数据
df = ak.macro_china_land()
# 土地溢价率
df = ak.macro_china_land_premium()
```

---

### 4. RPR 数据 - 租售比

**计算公式**: `租售比 = 房价 / 月租金 (单位: 月)`

| 优先级 | 数据源 | 类型 | 费用 | 说明 |
|--------|--------|------|------|------|
| ⭐⭐⭐⭐ | **AKShare** | 开源 | 免费 | 租金和房价指数 |
| ⭐⭐⭐ | **中国房价行情网** | 商业 | 免费/付费 | creprice.cn |
| ⭐⭐⭐ | **贝壳研究院** | 商业 | 免费/付费 | 租房数据 |
| ⭐⭐ | **58同城/安居客** | 商业 | 免费 | 租金数据 |
| ⭐ | **链家研究院** | 商业 | 报告 | 城市数据 |

## 🛠️ AKShare 数据接口汇总

AKShare 是 Python 开源金融数据库，提供免费的中国房地产数据接口：

### 安装
```bash
pip install akshare
```

### 常用接口

| 接口函数 | 数据类型 | 更新频率 |
|----------|----------|----------|
| `macro_china_new_house()` | 商品房销售 | 月度 |
| `macro_china_real_estate_investment()` | 房地产投资 | 月度 |
| `macro_china_land()` | 土地成交 | 月度 |
| `macro_china_land_premium()` | 土地溢价率 | 月度 |
| `macro_china_house_price()` | 房价指数 | 月度 |
| `stock_individual_info_em()` | 个股信息 | 实时 |

## 📡 数据获取策略建议

### 推荐方案 (免费为主)

```
┌─────────────────────────────────────────────────────────┐
│                    数据获取流程                          │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   [AKShare] ──────> [国家统计局数据]                    │
│       │                                                │
│       ├────> 商品房销售面积 (ACI分子)                   │
│       ├────> 土地成交数据 (LPR)                        │
│       └────> 房价/租金指数 (RPR)                       │
│                                                         │
│   [财报数据] ────> [上市房企融资] (FPI)                 │
│       │                                                │
│       ├──── 东方财富 API                               │
│       ├──── 新浪财经 API                               │
│       └──── 巨潮资讯网 (手动/爬虫)                      │
│                                                         │
│   [报告补充] ────> [中指研究院/克而瑞]                   │
│       │                                                │
│       └──── 库存数据/土拍报告 (补充)                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 优先级

1. **首选**: AKShare (免费开源，程序化调用)
2. **备选**: 国家统计局 + 东方财富 + 新浪财经 (免费API)
3. **补充**: 中指研究院/克而瑞报告 (月度更新)
4. **兜底**: 手动输入/模拟数据

## 🔧 配置说明

### 环境变量 (可选)

```bash
# 如需使用付费数据源，可配置以下环境变量
export NBS_TOKEN="国家统计局API Token"
export CHOICE_TOKEN="东方财富Token"
export WIND_TOKEN="Wind Token"
```

### 使用方式

```bash
# 使用模拟数据 (默认)
python src/data_fetchers/run_all.py

# 使用真实数据
python src/data_fetchers/run_all.py --real

# 获取租售比数据
python src/data_fetchers/run_all.py --rent
```
